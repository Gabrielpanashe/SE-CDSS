# SE-CDSS

**Sentiment-Enhanced Clinical Decision Support System for Precision Medicine** — undergraduate dissertation project . This software **assists** clinicians only; it does not replace professional judgment.

## Disclaimer

All API and UI surfaces that expose risk scores or recommendations include:

> This output is generated by an AI-assisted decision support system. It is advisory only. The final clinical decision must always be made by a qualified healthcare professional.

## Stack

- Python 3.10+
- FastAPI + Uvicorn (REST API)
- SQLAlchemy + SQLite (default; PostgreSQL later)
- scikit-learn baseline (TF-IDF + logistic regression)
- NLTK (tokenization, lemmatization, stopwords)
- Streamlit (optional UI; see [frontend/app.py](frontend/app.py))

## Repository layout

- [src/config.py](src/config.py) — paths, thresholds, `DRUG_MAP`, guideline scores, disclaimer text
- [src/preprocessing/](src/preprocessing/) — text pipeline and batch CSV preprocessing
- [src/models/](src/models/) — training and `predict()`

## Setup

```bash
python -m venv .venv
.venv\Scripts\activate   # Windows
pip install -r requirements.txt
```

Copy `.env.example` values as needed (or create `.env`):

- `DATABASE_URL=sqlite:///./se_cdss.db`

Train baseline (requires UCI Drug Review raw CSVs under `data/raw/`):

```bash
python -m src.preprocessing.preprocess
python -m src.models.base_line_model
```

Artifacts are written to `models/baseline/` (`model.pkl`, `vectorizer.pkl`). These files are gitignored; train locally or restore artifacts.

## Run API

From project root:

```bash
uvicorn api.main:app --reload
```

- OpenAPI: <http://127.0.0.1:8000/docs>
- Health: `GET /health`

### Main endpoints

- `POST /api/feedback` — review text → sentiment, risk, probabilities, `log_id`, disclaimer
- `GET /api/recommend` — query `condition`, `patient_id`, `sentiment` → top 3 drugs + disclaimer
- `GET /api/trends/{patient_id}` — prediction history + disclaimer

## Run Streamlit UI

Start the API first. Then:

```bash
set API_BASE_URL=http://127.0.0.1:8000
streamlit run frontend/app.py
```
